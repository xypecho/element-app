# vue版饿了么

>初学vue跟着视频写的vue版饿了么

## 技术栈
>vue2 + vue-cli + vue-resource + vue-router + webpack + ES6

## 项目运行
~~~~
git clone https://github.com/xypecho/element-app.git
npm install
npm run dev
~~~~
